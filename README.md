# Fitness

source: https://www.reddit.com/r/Fitness


## Diet
TW - Target Weight *(in pounds)*
GC - Goal Calorie

- Protein
  - Min: (0.8 x TW) grams
  - Max: (1 x TW) grams
  
- Fat
  - Min: (0.4 x TW) grams
  - Max: (0.5 x TW) grams
  
- Carbohydrates
  = (GC - ((P x 4) + (F x9)) / 4
  - `P` - Protein goal
  - `F` - Fat goal
